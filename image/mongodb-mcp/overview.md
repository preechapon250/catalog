## About MongoDB MCP Server

The MongoDB MCP Server is a Model Context Protocol implementation that enables AI applications to interact with MongoDB
databases. It provides tools for querying collections, discovering schemas, and performing database operations through
the MCP interface, allowing AI agents to work seamlessly with MongoDB data.

For more details, visit https://github.com/mongodb-js/mongodb-mcp-server.

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted

### Why use Docker Hardened Images?

These images are published with near-zero known CVEs, include signed provenance, and come with a complete Software Bill
of Materials (SBOM) and VEX metadata. They're designed to secure your software supply chain while fitting seamlessly
