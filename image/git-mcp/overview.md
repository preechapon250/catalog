## About Git MCP Server

The Git MCP Server is a Model Context Protocol implementation that provides tools to read, search, and manipulate Git
repositories programmatically via LLMs. It enables AI applications to perform Git operations such as reading files at
specific refs, listing commits, searching repository contents, and executing basic Git commands safely through the MCP
interface. This image includes Git and Git LFS support.

For more details, visit https://github.com/modelcontextprotocol/servers.
