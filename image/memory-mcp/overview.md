## About Memory MCP Server

The Memory MCP Server is a Model Context Protocol implementation that provides ephemeral and persistent memory tools for
MCP clients. It enables AI applications to store and retrieve contextual information across sessions, maintaining state
and context for enhanced conversational capabilities.

For more details, visit https://github.com/modelcontextprotocol/servers.
