## About Piraeus CSI

Piraeus CSI is a Container Storage Interface (CSI) driver for LINSTOR/Piraeus that enables Kubernetes to provision
persistent volumes using LINSTOR storage. It provides dynamic volume provisioning, snapshots, volume expansion, and
topology-aware scheduling for stateful workloads. The CSI driver integrates with the Kubernetes storage subsystem to
deliver replicated block storage powered by DRBD technology.
